🖨️ Printing Settings

Use separate width/font/auto-print options for the customer bill and the kitchen KOT.

  1. When the print dialog opens: Margins → None, Scale → 100% / Default, uncheck Headers and footers.
  2. In Chrome Destination, pick your 80mm printer and set paper size to 80mm / 72mm Roll if listed.
  3. Use each width below to match the correct printer/roll.

📲 RawBT Thermal Printing (Android)

Install the free RawBT app from Google Play on the Android device, pair your 58mm/80mm Bluetooth or USB thermal printer in RawBT once. Then turn these on to print silently — no Chrome print dialog. The UPI QR code prints as real text (no image needed).

QR needs UPI ID filled in Restaurant Info. RawBT buttons open the RawBT app — allow it the first time.

☁️ Live Multi-Device Sync (Firebase)

Enter your Firebase project config to sync data across ALL devices in real time. Get this from Firebase Console → Project Settings → General → Your apps → Web app.

Connecting…

Use this if you updated the menu/prices on this device and other devices still show the old menu. It makes THIS device's menu the one everyone gets.

First time setup: On your MAIN device, enter Firebase config, click Save then Upload My Data. On every OTHER device, enter the same config and click Save — it pulls everything automatically. After that it stays in sync in real time.
